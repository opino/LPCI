# LPCI
Stupid Questions

Sie wollen herausbekommen, von welchen Bibliotheken (Librarys) die Shell bash abhängt. Welches Kommando wird Ihnen hier helfen? 

- ldconfig /bin/bash
- ldd /var/bash
- cat ld.so.conf
- ldd /bin/bash
- cat ld.so.cache
